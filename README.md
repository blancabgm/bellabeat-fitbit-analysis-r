# Bellabeat Fitbit Analysis — R Project

Este proyecto analiza datos de actividad, sueño e intensidad de usuarios de Fitbit para identificar patrones aplicables a la estrategia de marketing de Bellabeat.

## 📁 Estructura del repositorio
- scripts/ → Código en R (limpieza, EDA, análisis avanzado)
- outputs/ → Gráficos exportados
- images/ → Visualizaciones del EDA
- README.md → Documentación del proyecto

## 🛠️ Herramientas
- R (tidyverse, lubridate, ggplot2)
- Posit Cloud
- Kaggle (dataset original)

## 📊 Contenido del análisis
- Limpieza completa de datasets (diario, sueño, horario, frecuencia cardíaca)
- EDA básico: distribuciones, relaciones, patrones horarios
- EDA avanzado: correlaciones, análisis por usuario, patrones semanales
- Recomendaciones accionables para Bellabeat

## 🔗 Dataset original
Fitbit Fitness Tracker Data (Kaggle)
